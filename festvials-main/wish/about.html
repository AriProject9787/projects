About Birthday Countdown

Birthday Countdown is a simple and fun web app that helps you remember and celebrate the birthdays of your friends and loved ones. Instead of forgetting important dates, this app ensures you’re always prepared with reminders and countdowns.

The app is designed with a clean interface, secure login, and privacy options for displaying your date of birth only if you choose to. Whether you’re tracking your best friend’s birthday or your entire family’s, this tool makes it easy.

✨ Key Features

  • 🔔 Automatic birthday reminders
  • 📅 Countdown timer for upcoming birthdays
  • 🔑 Secure login with privacy options
  • 📊 Personal dashboard to manage birthdays
  • 🌐 Responsive design for mobile and desktop

⚙️ Technologies Used

  • HTML5, CSS3, JavaScript – Core front-end structure
  • Firebase – User authentication & birthday data storage
  • AOS (Animate on Scroll) – Smooth animations
  • Responsive Web Design – Works on desktop & mobile
